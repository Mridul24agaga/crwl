Web Scraper API Documentation

Endpoint

POST https://dynamic-pegasus-0130a0.netlify.app/api/crawl

Request Format

{
    "url": "https://example.com",
    "max_pages": 5  // Optional, defaults to 10
}

Response Format

[
    {
        "metadata": {
            "url": "https://example.com",
            "title": "Page Title",
            "meta_description": "Page description"
        },
        "sections": {
            "header": {
                "navigation": [...],
                "text": "Header text"
            },
            "hero": {
                "title": "Hero title",
                "description": "Hero description",
                "buttons": [...]
            },
            // Other sections...
        }
    }
]

Example Usage

fetch('https://dynamic-pegasus-0130a0.netlify.app/api/crawl', {
    method: 'POST',
    headers: {
        'Content-Type': 'application/json',
    },
    body: JSON.stringify({
        url: 'https://example.com',
        max_pages: 5
    })
})
.then(response => response.json())
.then(data => console.log(data));
